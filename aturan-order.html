Keterangan Lengkap

  • Diskusikan dengan kami tentang apa yang Anda inginkan dengan website Anda agar kami dapat memberi arahan dan juga masukan. Bahkan kami lebih senang jika calon klien memberi contoh website yang diinginkan dengan cara mengirim url atau alamat situsnya. Jika tidak punya contoh, maka kami akan menawarkan tema website yang sesuai dengan keinginan claon klien tersebut.
  • Kami tidak menerima pesanan website yang melanggar hukum atau bertentangan dengan norma sosial seperti ; website penipuan, situs phising, situs pornografi, ujaran kebencian, judi, situs jual beli barang terlarang, situs kelompok/organisasi yang meresahkan dan sejenisnya.
  • Pengerjaan website akan dimulai setelah pembayaran uang muka. Uang muka tersebut kami gunakan untuk pembelian domain untuk website Anda. Namun data akun untuk pembelian domain tersebut akan kami berikan kepada klien jika pembayaran telah lunas. Jika klien tidak mau membayar sisa kekurangannya, maka kami akan mengosongkan isi web tersebut.
  • Data akun dan file backup dari semua klien selalu kami simpan dengan baik untuk mengantisipasi klien kehilangan file dan lupa data login. Namun kami tidak akan merubahnya kecuali ada permintaan dari klien kami.
  • Selalu bicarakan dengan kami terlebih dahulu jika Anda ingin merubah data atau konten sekecil apapun, karena kami tidak mau bertanggung jawab jika kesalahan dilakukan oleh klien tanpa memberi informasi kepada kami terlebih dahulu. "Jangan pernah melakukan sesuatu yang Anda tidak ketahui akibatnya".
  • Pengerjaan website kami lakukan umumnya dari pagi sampai sore hari. Lama pengerjaan web tergantung pada tingkat kesulitan pembuatan situs tersebut. Untuk situs profil perusahaan biasanya kami membutuhkan waktu 3 hari.
  • Revisi atau perbaikan konten gratis asal tidak ada unsur main-main dan tidak mengubah banyak tatanan dari template web tersebut. Kami tidak akan menanggapi permintaan klien yang hanya bertujuan mengerjai kami dengan cara revisi yang berulang-ulang. "Waktu lebih berharga daripada apapun".
  • Untuk memeriksa hasil pengerjaan, klien dapat langsung memeriksa dengan mengklik domain yang telah dipesan sebelumnya.
